## 핵심 기능
- **`인기리스트` :** 실제 유튜브사이트 내 Top10 영상 확인기능
- **`뮤직플레이어 기능` :** 메인페이지 우측 뮤직 플레이어 기능으로 영상 시간 조절 기능 구현
- **`영상 검색 기능` :** 좌측 사이드바를 통한 검색 기능 구현
